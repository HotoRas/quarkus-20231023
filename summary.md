## 기술 스택 분석

### 프론트엔드
- **마크업**: HTML5
- **스타일**: CSS3 (모듈화된 파일 구조)
- **스크립팅**: Vanilla JavaScript (외부 파일)
- **UI 요소**: 모달, 내비게이션 바, 토글러, Toast 알림

### 백엔드
- **프레임워크**: Quarkus (Java 기반)
- **데이터베이스**: SQLite / MySQL
- **API**: REST API 구조
- **클래스 구조**: Entity, Resource, Service 계층 분리
